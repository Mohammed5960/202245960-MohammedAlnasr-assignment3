# Personal Portfolio Website – Assignment 3

## Project Overview
This project is an advanced version of my portfolio website developed for Assignment 3. It builds on the previous assignments by adding API integration, more complex logic, and better state management.

The goal of this project is to create a more dynamic and interactive web application while applying best practices in JavaScript and web development.

---

## Features

### 1. Dynamic Project Section
- Live search functionality
- Filter projects by category (Web, JavaScript, Form)
- Sort projects (A-Z, Z-A, Newest, Oldest)

### 2. GitHub API Integration
- Fetches my public repositories using GitHub API
- Displays:
  - Repository name
  - Description
  - Programming language
  - Star count
- Handles:
  - Loading state
  - Error messages
  - Empty data

### 3. State Management
- Dark/light mode saved using localStorage
- Visitor name is saved and reused on next visit

### 4. Advanced Form Validation
- Checks:
  - Name is not empty and has at least 2 characters
  - Email format is valid
  - Message is at least 10 characters
- Shows clear error messages for each field

### 5. User Experience Enhancements
- Smooth transitions and hover effects
- Clear feedback messages for user actions
- Responsive design for mobile and desktop

---

## Technologies Used
- HTML5
- CSS3 (Flexbox, Grid, transitions)
- JavaScript (DOM manipulation, API calls, localStorage)
- GitHub API

## User Guide

- Enter your name in the top section and click "Save Name"
- Use the search bar to find specific projects
- Use the filter dropdown to show categories
- Use the sort menu to reorder projects
- GitHub repositories will load automatically
- Fill the contact form with valid inputs to send a message
- Toggle dark mode using the button in the navbar
